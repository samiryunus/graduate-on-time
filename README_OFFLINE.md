
# Transfer-to-Grad Planner (OFFLINE VERSION)

✅ No internet required  
✅ All processing happens locally  
✅ Safe for FERPA-sensitive transcripts  

## How to run (IMPORTANT)
Because browsers block local file reads, you MUST use a local server.

### Windows / Mac / Linux
1. Open the folder
2. Run:
   python -m http.server 8000
3. Open:
   http://localhost:8000

## Notes
- PDF parsing works offline
- If a PDF is scanned (image-only), manually paste courses
- This is a planning tool, not official advising
